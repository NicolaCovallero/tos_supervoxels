tableTop_object_segmentation_test.cpp

Description: This program tests the tos_supervoxels Library API serving it as a test. The table must be the bigger plane in the point cloud. It also shows how to retrieve the objects' points in the RGB image. The RGB image is built from the organized input cloud.

Usage:

$ ./bin/tableTop_objects_segmentation_test

to see the help with all the options you can use

$ ./bin/tos_supervoxels_test ./cloud.pcd

to segment the tabletop objects in the point cloud "cloud.pcd"

$ ./bin/tos_supervoxels_test ./cloud.pcd -smooth 5 -ct 7

To segment changing some parameters
